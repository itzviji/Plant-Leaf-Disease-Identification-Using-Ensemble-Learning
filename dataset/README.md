# 🌿 PlantVillage Dataset

## Overview

This project utilizes the **PlantVillage Dataset**, one of the most widely used benchmark datasets for plant leaf disease identification and classification. The dataset contains high-quality images of both healthy and diseased plant leaves, making it suitable for training and evaluating deep learning models for agricultural disease detection.

The dataset was used to train and validate the proposed **Plant Leaf Disease Identification and Classification System** based on **ResNet50**, **Ensemble Learning**, and the **Mish Activation Function**.

# 🧠 Why PlantVillage?

The PlantVillage dataset was selected because it:

- Contains a large number of annotated plant leaf images.
- Covers multiple crop species and disease categories.
- Is widely accepted as a benchmark dataset for plant disease classification.
- Provides high-quality images suitable for deep learning models.
- Enables fair comparison with existing research.

---

# 🔬 Usage in This Project

The dataset was used for:

- Image preprocessing
- Feature extraction using **ResNet50**
- Model training
- Model validation
- Performance evaluation
- Ensemble Learning classification

The trained model was further integrated into both a **React Native mobile application** and a **web application** for real-time disease prediction.
